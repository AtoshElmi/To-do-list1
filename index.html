    About This Project

    A modern, responsive to-do list application with local storage persistence and intuitive filtering.

    Key Features

    Add, complete, and delete tasks with a clean interface
    Filter tasks by status (All, Active, Completed)
    Tasks are saved to local storage automatically
    Fully responsive design for all devices

    How It Works

    • New tasks appear in "All" and "Active" tabs
    • Completed tasks move to "Completed" tab
    • Clear all completed tasks at once
    • Tasks count updates automatically

    Technology Used

    HTML5, CSS3, JavaScript (ES6), LocalStorage API
